Passphrase Generator

Download this project as a .zip file Download this project as a tar.gz file

This phrase consists of six common words chosen securely at random. It should be easy to remember but hard to guess, even for a super-computer, making it a good password.



Inspired by the following xkcd comic strip:

More information

  • Randomness is from the cryptographically secure getRandomValues function
  • All computation is done on the local computer's browser using javascript, no passphrase data is sent or received from any server
  • Words are chosen from a list of 2087 common words. This list was derived from the 10,000 most common passwords which were words in a dictionary but not swear words or proper nouns.
  • Source code available in main.js. Code is purposefully simple to allow easy verification.
